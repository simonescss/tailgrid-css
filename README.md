# TailGrid CSS
Layout Grid System based on Grid CSS
--
Demo Page: https://tailgrid.netlify.app \
Docs: (coming soon)

## Features
- New Grid System Span Method
- No Column Layout
- GSAP Animation

## Usage

### CSS
```html
<head>
  <link rel="stylesheet" href="tailgrid.min.css">
</head>
```
